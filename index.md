The USGS decommissioned their WaterWatch web pages in Febrary 2026. Direct replacements have not been provided for all pages. For example, the 1, 7, 14, and 28 day streamflow percentile maps do not have a direct replacement. The maps below are intended to recreate the functionality of those web pages using updated USGS tools. [This USGS tutorial](https://doi-usgs.github.io/hyswap/examples/site_conditions_examples.html) was used  as a reference for developing the Python code to download and process data and create the maps below. 

This is intended to be a short term solution until a more robust long term solution is developed.
